# Arithmetic Arranger

**Arithmetic Arranger** es una función en Python que organiza y formatea problemas de aritmética para una presentación clara y alineada. Esta herramienta facilita la visualización de operaciones aritméticas con operandos y resultados alineados, permitiendo una revisión más efectiva de problemas matemáticos simples.

## 🚀**Características**

- **Validación de Entradas**: Asegura que los números sean dígitos y no excedan los cuatro dígitos.
- **Formato de Salida**: Organiza los operandos, signos y resultados en un formato alineado.
- **Opcional**: Permite mostrar los resultados de las operaciones junto con las expresiones aritméticas.

## 💻 **Uso**

Para utilizar la función, simplemente llámala con una lista de problemas de aritmética y un parámetro opcional para mostrar respuestas:

```python
print(arithmetic_arranger(["323 + 8", "1 - 3801", "9999 + 9999", "523 - 49", "2 + 2000"], True))

📦 Instalación
No requiere instalación adicional. Solo necesitas Python 3.x.
